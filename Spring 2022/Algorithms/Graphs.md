# Terminology
* A graph G = (V,E) consists of:
	* A set of vertices V
	* A set of edges E
* Graphs with a few edges are **sparse**
* Graphs with lots of edges are **dense**
* Graphs with all possible edges are **complete**

A **cycle** is a loop of length >= 3
